# 利用規約
これはTRPGのシナリオ配布用HTML形式テンプレートです。
それ以外の用途にも、以下「NG 禁止事項」に反しない限り、ご利用いただけます。

本テンプレートは、HTML形式のシナリオ配布がもっとお気軽にできるようになるといいなあ、の考えのもと作られています。
（使用時には、ツール使用でHTML形式に変換してテンプレに流し込むとかすると楽だと思います。）

プログラム知識の少ない人間の作った拙いものですので、色々とお目こぼしいただけますと幸いです。

以下のサンプルサイトで挙動確認いただけます。




## NG 禁止事項
- 本テンプレートの自作発言
- 頒布ファイルそのままの形での再配布（無償/有償問わず）
- 法律に違反することに関連した利用



## OK 許可していること
### 購入不要、無料利用可能
 - 本テンプレートを利用した【シナリオ】の公開及び頒布
   - 無償/有償問わず
   - 改変した場合を含む
   - 既存のHTMLファイルに部分利用した場合を含む

### 「テンプレート配布用」の購入が必要
- 本テンプレートを改変・部分利用した【テンプレート】の公開及び配布
   - 無償/有償問わず
   - 改変・部分利用の大小を問わず
   - 配布時には、配布物内に本テンプレートの配布元情報記載が必要。要リンクシェア。
